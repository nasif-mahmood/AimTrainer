# Aim Trainer

A Python-based aim training game designed to improve your mouse accuracy and reaction time.

## 🎮 Gameplay Screenshots

### Game Demo
![Game Demo](https://github.com/user-attachments/assets/c0601a36-1a26-4bb5-bf57-7de7e583550d)

## 🛠️ Technologies & Tools

- **Python 3** - Core programming language
- **Pygame** - Game engine and graphics rendering

## 📋 Features

- Real-time accuracy tracking
- Reaction time measurement
- Score display
- End game statistics

## 🚀 Getting Started

### Prerequisites
- Python 3.x
- Pygame

### Installation

```bash
pip install pygame
```

### Running the Game

```bash
python main.py
```
